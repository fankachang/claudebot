開始使用

一行指令就能啟動

推薦：一鍵安裝 
npx claudebot-app

或手動安裝：

# Clone & install
git clone https://github.com/Jeffrey0117/ClaudeBot.git
cd ClaudeBot && npm install

# Interactive setup wizard
npm run setup

# Launch
npm run dev

需要：Node.js 20+、Claude CLI（已登入）。選配：Gemini CLI、ffmpeg、Python 3.11+。
