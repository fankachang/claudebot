核心功能

打造最強 Telegram AI 指揮中心

即時串流

每秒更新回應狀態、工具名稱、執行耗時，掌握 AI 每一步動態。

多 AI 後端

支援 Claude / Gemini / auto 模式，/model 一鍵切換，選擇最適合的 AI。

多專案管理

每個專案獨立 Session，--resume 延續對話，跨專案無縫切換。

佇列系統

序列執行 + 跨 Bot 檔案鎖，確保一次只有一個 CLI 進程運行。

互動式 UI

確認按鈕、建議追問、選項偵測，讓 AI 互動不只是文字。

語音辨識

Sherpa-ONNX 本地語音辨識 + 2x 加速播放 + LLM 智慧修正。

插件系統

熱重載、零 AI 成本、Plugin Store 一鍵安裝，輕鬆擴展功能。

自動提交

AI 改動後自動 git commit + push，程式碼版本控制全自動。

Web 儀表板

React 面板 + WebSocket 即時監控，視覺化掌握 Bot 狀態。

開始使用

複製、設定、啟動

# Clone & install
git clone https://github.com/Jeffrey0117/ClaudeBot.git
cd ClaudeBot && npm install

# Configure
cp .env.example .env
# Fill in: BOT_TOKEN, PASSWORD, ALLOWED_PROJECTS

# Launch
npm run dev
